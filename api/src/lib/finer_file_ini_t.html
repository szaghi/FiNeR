load ​

Get file data from a file or a source string.

Usage ​

Loading from a file ​
bash
type(file_ini):: fini
call fini%load(filename='path_to_my_file.ini')
Loading from a source string ​
bash
type(file_ini):: fini
call fini%load(source='[section-1] option-1=one [section-2] option-2=due')
